2. 风机布置

根据风电场的风能分布情况并结合地形等其他影响因素，
按尽可能利用风能、满足施工运输、缩短集电线路及节省土地
等布置原则, “修改报告”在保持总装机容量基本不变的情况
下，对初选机型分别进行布置，具体机型布置方案为:

方案工: 布置25 合单机容量 2000kW ( WTG1) 风力发电
机组，装机容量 SOMW;

方案芝: 布置23 台单机容量 2000kW ( WTG2 ) 风力发电
机组，装机容量 30MW;

方案II: 布置 25 台单机容量 2000kW ( WTG3 ) 风力发电
机组，装机容量 50MW;

方案IV: 布置 20 合单机容量 2500kKW ( WTG4 ) 风力发电
机组，装机容量 50MW;

方案V: 布置 20 人台单机容量 2500kW ( WTG5 ) 风力发电
机组，装机容量 30MW;

方案V: 布置 16 合单机容量 3000kW ( WTG6 ) 风力发电
机组，装机容量 48MW;
